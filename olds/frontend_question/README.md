## 20180116-js-两个面试题

- 第一题, 分析下面代码, 并给出结果
```javascript
    var a = 0, b = 0;
    function A( a ) {
        A = function ( b ) {
            alert( a + b++ );
        };
        alert( a++ );
    }
    A( 1 );
    A( 2 );
```
- 第二题, 分析下面代码, 并给出结果
```javascript
    var a = { num: 2 };
    var b = a;
    a.num = a = { num: 4 };
    console.log( a.num );
    console.log( b.num );
```
